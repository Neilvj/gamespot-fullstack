# GameSpot â€“ Fullstack Inventory Project

GameSpot is a beginner-friendly fullstack web application for managing a game inventory.
The project was built as a learning exercise to practice backend APIs, database relations,
and frontend CRUD interaction.

This repository represents a completed MySQL-based fullstack project, created before
rebuilding the backend with MongoDB and Mongoose for a separate assignment.

---

## Features

- Create, read, update and delete games
- Games can belong to multiple genres
- Games can be available on multiple platforms
- Edit games using the same form UI
- Filter by genre
- Sort by title or price
- Group by platform or genre
- Simple frontend search (title and developer)
- Platform badges in the game list
